## 教学小栈
+ 是一个什么样的项目
    + 该项目是一个基于Vue框架的教学信息管理系统，主要用途是方便老师管理自己的课程和学生信息。主要的功能有用户登录、学生信息管理、课程信息管理等。
+ 用到了什么技术
    + Ajax、axios
        + 在项目中用来向后端请求用户信息完成登录功能，请求学生信息完成学生管理功能。
    + Vuex
        + 使用vuex实现项目不同组件之间的数据共享
    + vue-router路由
        + 该项目主体相当于一个单页面应用，所以使用了vue-router
    + echart
        + 主要用于学生和课程相关数据的可视化
    + mock
        + 模拟后端
    + element-ui
        + 方便项目的搭建和开发
